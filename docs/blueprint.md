# **App Name**: Python Path Compass

## Core Features:

- Interactive Roadmap Navigator: Visually display the onboarding roadmap, allowing juniors to navigate between topics like Python installation, environments, OOPS, SQL, Docker, and Django, providing an overview of each step.
- Detailed Concept Explanations: Provide comprehensive, visually engaging pages for each roadmap topic, explaining core concepts such as 'why we build env', 'interpreter based syncronous lang behavior', 'SQL data storage and relations', and 'Docker containerization and compose file parameters'.
- Code and Command Snippet Viewer: Integrate interactive code and command snippets (e.g., Python installation, pip/uv usage, Dockerfiles, docker-compose.yml, Django commands), ensuring proper syntax highlighting and copy-to-clipboard functionality.
- AI Concept Clarifier Tool: A generative AI tool that can take user questions about any roadmap topic and provide simplified explanations, alternative analogies, or additional context suitable for a junior developer.
- External Reference & Resource Hub: Curate and link to essential external resources, including official documentation (e.g., Django docs), recommended video tutorials (like the specified YouTube link), and articles, organized by topic.
- Roadmap Content Management: Store all roadmap topics, detailed explanations, code snippets, and external resource links in a structured SQL database for easy updates and content management.

## Style Guidelines:

- Primary color: A clear and confident medium-dark blue (#4593D9), representing intelligence and reliability, suitable for a learning platform.
- Background color: A very light, almost white, desaturated blue (#EBEFF3), ensuring excellent readability for text and code on a clean canvas.
- Accent color: A refreshing and slightly brighter cyan (#7CCECE), providing visual emphasis for interactive elements and highlights, complementing the primary blue.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel in titles and key headings.
- Body font: 'Inter' (sans-serif) for high readability in longer text blocks and detailed explanations, offering a neutral and objective tone.
- Code font: 'Source Code Pro' (monospace sans-serif) specifically for clear, legible display of all code and command snippets, as well as file paths.
- Utilize a clean, responsive multi-column layout with a prominent sidebar for roadmap navigation, ensuring content is digestible and easy to follow on all device sizes.
- Implement subtle transition animations when navigating between roadmap topics and unfolding detailed content sections to enhance the user experience.
